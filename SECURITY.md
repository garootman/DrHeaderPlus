# Security Policy

## Supported Versions

DrHeaderPlus is a best-effort project. We will try to fix issues and welcome bug reports and feature requests.

| Version | Supported          |
| ------- | ------------------ |
| 3.x     | :white_check_mark: |

## Reporting a Vulnerability

If you find a vulnerability please raise an issue and let us know!

If you think you can fix it then feel free to put it in a branch and raise a pull request.
