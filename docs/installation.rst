============
Installation
============


Stable release
--------------

To install DrHeaderPlus, run this command in your terminal:

.. code-block:: console

    $ pip install drheaderplus

This is the preferred method to install DrHeaderPlus, as it will always install the most recent stable release from PyPI.

If you don't have `pip`_ installed, this `Python installation guide`_ can guide
you through the process.

.. _pip: https://pip.pypa.io
.. _Python installation guide: http://docs.python-guide.org/en/latest/starting/installation/

From sources
------------

The sources for DrHeaderPlus can be downloaded from the `Github repo`_.

You can either clone the public repository:

.. code-block:: console

    $ git clone git://github.com/garootman/DrHeaderPlus

Or download a zip file containing the current main branch:

.. code-block:: console

    $ curl  -OL https://github.com/garootman/DrHeaderPlus/archive/main.zip

Once you have a copy of the source, you can install it with:

.. code-block:: console

    $ pip install .


.. _Github repo: https://github.com/garootman/DrHeaderPlus/
